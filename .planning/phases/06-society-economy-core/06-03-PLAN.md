     - Click handler: switch active state, call _updateRankingChart(mode)
   - Canvas container: div with position:relative, height:400px
   - Canvas id: 'inequality-ranking-canvas'
   - Render horizontal bar chart:
     - type: 'bar'
     - indexAxis: 'y' (horizontal bars)
     - Initial data: INCOME_GINI sorted descending by gini
       - labels: country names
       - data: gini values
       - backgroundColor: per-bar color gradient based on gini:
         - >= 50: toRgba(CHART_COLORS.crisis, 0.7) (high inequality, red)

   - Store chart reference in _rankingChart
