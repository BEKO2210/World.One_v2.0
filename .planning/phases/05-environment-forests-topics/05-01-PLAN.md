**1. Create detail/utils/choropleth.js (INFRA-06):**

Extract and parameterize the SVG choropleth pattern from temperature.js (lines 663-812) into a reusable module. The function signature:

```javascript
export async function renderChoropleth(container, options)
```

Parameters:
- `container` (HTMLElement): Parent element to append map into
- `options.dataMap` (Object): ISO-2 code -> numeric value mapping (e.g., { DE: 0.38, FR: 0.33 })
- `options.colorFn` (Function): (value) => CSS color string
- `options.tooltipFn` (Function): (iso, value) => tooltip HTML text string
- `options.legendItems` (Array): [{ color, label }] for discrete legend swatches
- `options.title` (string, optional): Heading text above the map
- `options.defaultColor` (string, optional): Color for countries with no data, default 'rgba(255,255,255,0.05)'

Returns: `{ wrapper: HTMLElement, cleanup: Function }` -- cleanup removes event listeners.

Implementation details:
- Import DOMUtils from `../../js/utils/dom.js`
- Copy the full CLASS_TO_ISO mapping from temperature.js (80+ entries covering country name -> ISO-2 code)
- Use _basePath() pattern: `window.location.pathname.includes('/detail') ? '../' : ''` for SVG fetch
- Fetch `${basePath}assets/maps/world.svg`, inject via wrapper.innerHTML
- Style SVG: width 100%, height auto, preserveAspectRatio xMidYMid meet, remove default fill
- ISO resolution order: path.id (2-letter) -> CLASS_TO_ISO[className.baseVal or first class] -> CLASS_TO_ISO[name attribute]
- Color each path: if dataMap has ISO -> colorFn(value), else -> defaultColor
- TOOLTIP_STYLE constant matching temperature.js: position absolute, background rgba(18,18,26,0.9), border 1px solid rgba(255,255,255,0.08), borderRadius 8px, padding 8px 12px, color var(--text-primary), fontSize 0.8rem, pointerEvents none, zIndex 1000, whiteSpace nowrap
- Mouse handlers: mouseenter (show tooltip via tooltipFn, highlight path with brightness filter), mousemove (position tooltip near cursor with boundary clamping), mouseleave (hide tooltip, remove highlight)
- Legend: horizontal flex row of color swatches with labels below the map
- Fallback: if SVG fetch fails, append "Map unavailable" text (italic, secondary color)
- Track all event listeners for cleanup function
