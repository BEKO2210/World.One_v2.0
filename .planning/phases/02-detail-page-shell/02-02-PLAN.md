  <name>Task 1: Create detail-app.js with routing, module loading, error handling, and all UI controls</name>

  <action>
Create `detail/detail-app.js` as a self-executing ES module. This is the single entry point for the detail page.

**Imports (all paths relative to detail/ directory):**
```javascript
import { i18n } from '../js/i18n.js';
import { DOMUtils } from '../js/utils/dom.js';
import { destroyAllCharts, ensureChartJs, createChart } from '../js/utils/chart-manager.js';
```

**Module-level state:**
```javascript
let _currentTopic = null;  // reference to loaded topic module for cleanup
let _toastTimeout = null;  // for toast overlap prevention
```

**VALID_TOPICS allowlist** (DETAIL-02 security requirement -- must validate before import):
```javascript
const VALID_TOPICS = [
  'co2', 'temperature', 'biodiversity', 'airquality', 'forests',
  'renewables', 'population', 'conflicts', 'health', 'freedom',
  'inequality', 'poverty', 'currencies', 'science', 'internet',
  'space', 'weather', 'earthquakes', 'solar', 'crypto_sentiment',
  'momentum_detail', 'hunger', 'disasters', 'ocean_temp',
  'ocean_ph', 'ocean_plastic', 'extinction', 'endangered', '_stub'
];
```
Note: `_stub` is included for development/testing.

**Function: async init()** -- called at module load time:
1. Parse URL: `const params = new URLSearchParams(window.location.search)`, get `topic` and `lang`
2. If `lang` is 'de' or 'en', set `i18n.lang = lang`
3. Call `i18n.init()` to apply initial translations to DOM
4. If `!topic` or `!VALID_TOPICS.includes(topic)`, call `renderErrorState('invalid-topic', topic)` and return
5. Call `await loadTopic(topic)`
6. Call `setupNavControls()`
7. Call `setupBackToTop()`
8. Register cleanup: `window.addEventListener('beforeunload', cleanup)`

**Function: async loadTopic(topicId):**
1. Log `[DetailApp] Loading topic: ${topicId}`
2. Try:
   a. `const module = await import(\`./topics/${topicId}.js\`)`
   b. Validate contract: check `typeof module.render === 'function'` and `typeof module.cleanup === 'function'`. If missing, throw Error with descriptive message
   c. Set `_currentTopic = module`
   d. Build blocks object: `{ hero: document.getElementById('detail-hero'), chart: document.getElementById('detail-chart'), trend: document.getElementById('detail-trend'), tiles: document.getElementById('detail-tiles'), explanation: document.getElementById('detail-explanation'), comparison: document.getElementById('detail-comparison'), sources: document.getElementById('detail-sources') }`
   e. Call `await module.render(blocks)`
   f. Remove skeleton class from all blocks: `document.querySelectorAll('.detail-block--skeleton').forEach(el => el.classList.remove('detail-block--skeleton'))`
   g. Update breadcrumb: `document.getElementById('detail-breadcrumb-topic').textContent = i18n.t(module.meta.titleKey)` (fallback to topicId if meta.titleKey not found)
   h. Update page title: `document.title = \`World.One - ${i18n.t(module.meta.titleKey)}\``
   i. Setup lazy charts if module provides them: call `setupLazyCharts(module.getChartConfigs())`
   j. Add time range selector to the trend block if it exists: call `setupTimeRange(document.getElementById('detail-trend'))` -- insert the time range selector at the top of the trend block
   k. Log `[DetailApp] Topic loaded: ${topicId}`
3. Catch: Log error with `console.error('[DetailApp] Failed to load topic:', err)`, call `renderErrorState('load-failed', topicId)`

**Function: renderErrorState(type, topicId):**
1. Hide main: `document.getElementById('detail-main').hidden = true`
2. Show error: `document.getElementById('detail-error').hidden = false`
3. Set message based on type:
   - 'invalid-topic': `i18n.t('detail.error.invalidTopic', { topic: topicId || '?' })`
   - 'load-failed': `i18n.t('detail.error.loadFailed', { topic: topicId })`
4. Set message text on `document.getElementById('detail-error-message')`
5. Update page title: `document.title = \`World.One - ${i18n.t('detail.error.title')}\``

**Function: setupNavControls():**
1. Share button click handler (getElementById 'detail-share'):
   - Build shareData: `{ title: document.title, text: i18n.t('detail.shareText', { topic: i18n.t(_currentTopic?.meta?.titleKey || '') }), url: window.location.href }`
   - If `navigator.share && navigator.canShare && navigator.canShare(shareData)`: try `await navigator.share(shareData)`, catch and ignore AbortError (err.name !== 'AbortError' -> console.warn)
   - Else (clipboard fallback): try `await navigator.clipboard.writeText(window.location.href)`, then call `showToast(i18n.t('detail.copied'))`
   - Catch clipboard errors with console.warn('[DetailApp] Clipboard failed:', err)
2. Language toggle click handler (getElementById 'detail-lang-toggle'):
   - Call `i18n.toggle()`
   - Update the lang label span (querySelector '.detail-nav__lang-label'): if i18n.lang is 'de' show 'EN', if 'en' show 'DE'
   - If `_currentTopic`, update breadcrumb text: `document.getElementById('detail-breadcrumb-topic').textContent = i18n.t(_currentTopic.meta.titleKey)`
   - Update page title: `document.title = \`World.One - ${i18n.t(_currentTopic.meta.titleKey)}\``

**Function: setupBackToTop():**
1. Get BTT button by id 'detail-btt'
2. Create a throttled scroll handler (DOMUtils.throttle with 200ms delay):
   - If `window.scrollY > window.innerHeight`: `btt.removeAttribute('hidden')`
   - Else: `btt.setAttribute('hidden', '')`
3. Add scroll event listener with the throttled handler
4. On BTT click: `window.scrollTo({ top: 0, behavior: 'smooth' })`

**Function: setupLazyCharts(chartConfigs):**
- If `!chartConfigs || chartConfigs.length === 0`, return
- For each `{ canvasId, config, blockId }` in chartConfigs:
  - Get block element by blockId
  - If not found, continue
  - Use `DOMUtils.observe(block, (entry) => { ... }, { threshold: 0.1, rootMargin: '200px' })` where callback:
    - If `entry.isIntersecting`:
      - `ensureChartJs().then(() => { createChart(canvasId, config); })`

**Function: setupTimeRange(trendBlock):**
- If `!trendBlock`, return
- Create the time range selector using DOMUtils.create:
  - Container div with class 'time-range', role 'group', aria-label from i18n.t('detail.range.label')
  - 4 buttons: keys '1y', '5y', '20y', 'max' with text from i18n.t('detail.range.{key}')
  - 'max' button gets class 'time-range__btn time-range__btn--active', others get 'time-range__btn'
  - Click handler on each button:
    - Find current active: `container.querySelector('.time-range__btn--active')?.classList.remove('time-range__btn--active')`
    - Add active to clicked: `btn.classList.add('time-range__btn--active')`
    - Dispatch CustomEvent on trendBlock: `trendBlock.dispatchEvent(new CustomEvent('timerangechange', { detail: { range: key } }))`
- Prepend the container to trendBlock: `trendBlock.insertBefore(container, trendBlock.firstChild)`

**Function: showToast(message):**
- Get toast element by id 'detail-toast'
- Set textContent to message
- Remove hidden attribute: `toast.removeAttribute('hidden')`
- Clear any existing timeout: `if (_toastTimeout) clearTimeout(_toastTimeout)`
- Set new timeout: `_toastTimeout = setTimeout(() => { toast.setAttribute('hidden', ''); }, 2500)`

**Function: cleanup():**
- If `_currentTopic && typeof _currentTopic.cleanup === 'function'`, call `_currentTopic.cleanup()`
- Call `destroyAllCharts()`

**Self-execution at bottom of module:**
```javascript
init().catch(err => {
  console.error('[DetailApp] Init failed:', err);
  renderErrorState('load-failed', 'unknown');
});
```

Use console logging with `[DetailApp]` prefix throughout (project convention). All user-facing strings via `i18n.t()`. No hardcoded German or English strings.
