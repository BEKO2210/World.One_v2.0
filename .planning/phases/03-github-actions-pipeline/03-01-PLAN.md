  <files>scripts/cache-utils.js, data/cache/.gitkeep</files>
  <action>
1. Create `data/cache/.gitkeep` (empty file) so the cache output directory exists in git.

2. Create `scripts/cache-utils.js` as an ESM module (matching project `"type": "module"`) exporting these functions:

**fetchJSON(url, options)** -- HTTP fetch with retry and timeout:
- Parameters: `url` (string), `options` (object with `retries` default 2, `timeout` default 15000, `headers` default {})
- Uses AbortController for timeout (NOT AbortSignal.timeout -- that is browser-only; Node 20 supports it but AbortController is more portable across test environments)
- User-Agent header: `'BelkisOne/1.0 CachePipeline'`
- Retry with exponential backoff: `1000 * (attempt + 1)` ms delay between retries
- Throws on final failure after all retries exhausted
- Validates response is not null/undefined before returning

**fetchText(url, options)** -- Same retry pattern but returns `res.text()` instead of `res.json()`

**extractWorldBankEntries(data, options)** -- Extract and clean World Bank API v2 responses:
- Input: Raw World Bank response (array where [1] contains entries)
- Filters out null/undefined values
- Parses `e.date` as integer year
- Rounds `e.value` to `roundDigits` decimal places (default 1)
- Filters out NaN years and non-finite values
- Returns sorted array of `{ year, value }` objects

**saveCache(filename, data)** -- Write cache JSON file:
- Creates `data/cache/` directory recursively if missing (`mkdirSync(CACHE_DIR, { recursive: true })`)
- Wraps `data` in an object with `_meta: { fetched_at: new Date().toISOString(), source: 'cache-pipeline', version: '1.0' }` spread BEFORE the topic data (so `_meta` is always first in output JSON)
- Writes to `join(CACHE_DIR, filename)` using `JSON.stringify(output, null, 2)`
- Logs `"  OK {filename}"` to stdout on success

**CACHE_DIR** -- Exported constant: `join(__dirname, '..', 'data', 'cache')`

Use `import { writeFileSync, mkdirSync } from 'fs'`, `import { join, dirname } from 'path'`, `import { fileURLToPath } from 'url'` for ESM __dirname derivation.

All three scripts follow the same pattern: `#!/usr/bin/env node` shebang, ESM imports from `./cache-utils.js`, an async `main()` that calls API-specific fetch functions, wraps results in `saveCache()`, and catches top-level errors with `process.exit(1)`.

Each script logs a header line (e.g., `console.log('=== update-biodiversity ===')`) and a summary line at end (e.g., `console.log('Done: N files written')`).

**cache-biodiversity.js** (ACTIONS-02):
- Fetch GBIF threatened species: `https://api.gbif.org/v1/species/search?threat=VULNERABLE&limit=0` (count only), repeat for ENDANGERED, CRITICALLY_ENDANGERED
- Build object: `{ threatened_counts: { vulnerable: N, endangered: N, critically_endangered: N, total: sum }, timestamp: ISO }` (use the count property from GBIF response: `result.count`)
- `saveCache('biodiversity.json', data)`
- Wrap each GBIF fetch in try/catch so a single status failure doesn't crash the script; log warning and set count to null for failed statuses

**cache-environment-ext.js** (ACTIONS-03):
- **co2-history.json**: Fetch NOAA Mauna Loa monthly CSV from `https://gml.noaa.gov/webdata/ccgg/trends/co2/co2_mm_mlo.csv` via `fetchText()`. Parse CSV: skip comment lines starting with `#`, split remaining lines by whitespace or comma, extract columns: year (col 0), month (col 1), average (col 3 -- "interpolated" monthly mean). Build array of `{ year, month, value }` objects. Filter to valid numeric rows only. Save as `{ monthly: [...] }`
- **ocean.json**: Fetch NOAA Coral Reef Watch global SST from `https://coralreefwatch.noaa.gov/product/vs/data/all_vs_data.json`. If this specific endpoint fails (it may return non-JSON), fall back to a simpler approach: use fetchJSON on `https://www.ncei.noaa.gov/access/monitoring/climate-at-a-glance/global/time-series/globe/ocean/ytd/12/1880-2026.json` (NOAA Climate at a Glance). Extract the `data` object (keys are years, values are anomaly strings). Build array of `{ year, anomaly }`. Save as `{ annual_sst_anomaly: [...] }`
- **solar.json**: Fetch NOAA SWPC solar data from `https://services.swpc.noaa.gov/json/solar-cycle/predicted-solar-cycle.json` via `fetchJSON()`. Extract recent entries (last 36 months). Save as `{ solar_cycle: [...last 36 entries] }`
- `saveCache()` for each file separately -- 3 calls total

**cache-society-ext.js** (ACTIONS-04):
- **population.json**: Fetch World Bank `SP.POP.TOTL` for world (`WLD`): `https://api.worldbank.org/v2/country/WLD/indicator/SP.POP.TOTL?format=json&per_page=60&date=1960:2025`. Use `extractWorldBankEntries()`. Also fetch urban population `SP.URB.TOTL.IN.ZS` same pattern. Save as `{ total: [...], urban_percent: [...] }`
- **freedom.json**: Freedom House has NO API. Create a pre-processed static dataset directly in the script as a const array of `{ year, score }` covering 2006-2025 global freedom scores (the aggregate "world" score from their annual reports: 2006: 47.0, trending down to ~43.0 by 2025). This is acceptable because the data only updates once/year. Save as `{ global_trend: [...] }`
- **conflicts.json**: UCDP API now requires a token. Attempt fetch from `https://ucdpapi.pcr.uu.se/api/gedevents/24.1?pagesize=1&page=1` with optional `x-ucdp-access-token` header from `process.env.UCDP_API_TOKEN`. If that fails (no token or 401/403), fall back to a pre-processed static summary: `{ active_conflicts: 56, year: 2024, source: 'UCDP/PRIO Armed Conflict Dataset v24.1', note: 'Static fallback -- UCDP API requires token since Feb 2026' }`. Log which path was taken. Save as `{ conflict_data: {...} }`
- `saveCache()` for each file -- 3 calls total
