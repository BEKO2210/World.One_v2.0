**CDN Loader:**

Module-level state:
```javascript
let _chartJsLoaded = false;
let _chartJsLoading = null;
```

Export `ensureChartJs()` -- returns a Promise that resolves when Chart.js is ready:
1. If _chartJsLoaded is true, return Promise.resolve() immediately.
2. If _chartJsLoading is not null, return the existing promise (singleton guard against race conditions).
3. Otherwise, set _chartJsLoading to a new promise chain:
   - Try primary CDN: `_loadScript('https://cdn.jsdelivr.net/npm/chart.js@4.5.1/dist/chart.umd.min.js')`
   - On failure, try fallback: `.catch(() => _loadScript('https://cdnjs.cloudflare.com/ajax/libs/Chart.js/4.5.0/chart.umd.min.js'))`
   - On success: set _chartJsLoaded = true, call _applyDarkDefaults(), log `console.log('[ChartManager] Chart.js loaded')`
   - On total failure: log `console.error('[ChartManager] Failed to load Chart.js from all CDNs')`, reset _chartJsLoading = null, re-throw
4. Return _chartJsLoading.

Private helper `_loadScript(src)`:
- Creates a script element, sets src, appends to document.head
- Returns a Promise that resolves on script.onload, rejects on script.onerror
- Log attempt: `console.log('[ChartManager] Loading:', src)`

**Instance Registry:**

Module-level state:
```javascript
const _instances = new Map();
```

Export `createChart(canvasId, config)`:
1. First call `destroyChart(canvasId)` to clean up any existing chart on same canvas (prevents memory leak).
2. Get canvas element: `document.getElementById(canvasId)`. If not found, log `console.warn('[ChartManager] Canvas not found:', canvasId)` and return null.
3. Create: `const chart = new Chart(ctx, config)`.
4. Track: `_instances.set(canvasId, chart)`.
5. Log: `console.log('[ChartManager] Created chart:', canvasId)`.
6. Return chart instance.

Export `destroyChart(canvasId)`:
1. Get from registry: `_instances.get(canvasId)`.
2. If exists: call `existing.destroy()`, delete from map, log `console.log('[ChartManager] Destroyed chart:', canvasId)`.

Export `destroyAllCharts()`:
1. Log: `console.log('[ChartManager] Destroying all charts:', _instances.size, 'instances')`.
2. Iterate: `for (const [id, chart] of _instances) { chart.destroy(); }`.
3. Clear: `_instances.clear()`.

IMPORTANT: createChart does NOT call ensureChartJs() itself. The caller must await ensureChartJs() before calling createChart(). This keeps the registry synchronous and predictable. Document this in a comment above createChart.

Follow conventions: named exports, underscore-prefixed private state, [ChartManager] log prefix, decorative comment separators between CDN loader section, defaults section, and registry section. 2-space indentation, single quotes.
