**data-loader.js path fix:**

The data-loader uses relative paths (`data/cache/${topic}.json` and `data/fallback/static-values.json`) that resolve incorrectly from the `detail/` subdirectory. The detail page HTML at `detail/index.html` loads scripts with `../js/` prefix, but fetch() resolves relative to the HTML document URL, not the script URL.

Add a `_basePath()` helper that detects if the current page is in a subdirectory and returns the appropriate prefix. Implementation:

```javascript
function _basePath() {
  // If URL path includes /detail/, we need ../ to reach root
  return window.location.pathname.includes('/detail') ? '../' : '';
}
```

Update the two fetch calls in `fetchTopicData()`:
- Cache URL: change `data/cache/${topic}.json` to `${_basePath()}data/cache/${topic}.json`
- Static fallback URL in `_getStaticFallback()`: change `data/fallback/static-values.json` to `${_basePath()}data/fallback/static-values.json`

Do NOT change the function signatures or return types. Do NOT change `fetchWithTimeout`.

Use hardcoded data (World Bank API is supplementary, not primary per research -- "Hardcoded array" is the fallback). 6. **Tiles block** (`blocks.tiles`): 2x2 grid (matching _stub.js pattern). Four tiles:
   - Annual increase: 2.4 ppm/yr (from static or computed from last 12 months of monthly data)
   - Total emissions: 37.4 Gt CO2 (from static fallback)
