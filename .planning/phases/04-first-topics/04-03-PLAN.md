   - 24h data: `fetchWithTimeout('https://earthquake.usgs.gov/earthquakes/feed/v1.0/summary/2.5_day.geojson', 8000)`
   - 7d data: `fetchWithTimeout('https://earthquake.usgs.gov/earthquakes/feed/v1.0/summary/2.5_week.geojson', 8000)`

   Parse GeoJSON: `features.map(f => ({ lat: f.geometry.coordinates[1], lng: f.geometry.coordinates[0], depth: f.geometry.coordinates[2], magnitude: f.properties.mag, place: f.properties.place, time: f.properties.time }))`.

   Then plot earthquake dots. For each quake in 24h data:

   Add click handler on each circle to show popup. The popup is an absolutely-positioned div (NOT SVG -- use HTML overlay for better text rendering):
   ```javascript
   function showPopup(container, quake) {
     if (_tooltipEl) _tooltipEl.remove();
     _tooltipEl = DOMUtils.create('div', {
       className: 'quake-popup',
       style: {
         position: 'absolute',
         background: 'rgba(18, 18, 26, 0.95)',
         border: '1px solid rgba(255, 255, 255, 0.1)',
         borderRadius: '8px',
         padding: '10px 14px',
         color: 'var(--text-primary)',
         fontSize: '0.85rem',
         zIndex: '10',
         pointerEvents: 'none',
         maxWidth: '250px',
       }
     }, [
       DOMUtils.create('div', { textContent: `M${quake.magnitude.toFixed(1)}`, style: { fontWeight: '700', fontSize: '1.1rem', marginBottom: '4px' } }),
       DOMUtils.create('div', { textContent: quake.place, style: { marginBottom: '4px' } }),
       DOMUtils.create('div', { textContent: `${i18n.t('detail.earthquakes.depth')}: ${quake.depth.toFixed(1)} km`, style: { color: 'var(--text-secondary)' } }),
       DOMUtils.create('div', { textContent: new Date(quake.time).toLocaleString(), style: { color: 'var(--text-secondary)', fontSize: '0.75rem' } }),
     ]);
     container.appendChild(_tooltipEl);
   }
   ```

   Position the popup near the clicked circle. The container must have `position: relative` style. On click outside or on a different circle, remove the previous popup.

   Add a map legend below the SVG showing depth color scale: 4 colored circles with labels (Shallow < 10km, Medium 10-70km, Deep 70-300km, Very Deep > 300km).

4. **Trend block** -- **7-Day Magnitude Histogram**: Create canvas `quake-histogram-canvas`. Build histogram with 5 bins: 2.5-3.0, 3.0-4.0, 4.0-5.0, 5.0-6.0, 6.0+. Count 7-day quakes into bins. Chart.js bar chart with `CHART_COLORS.realtime` color. See research Example 4 for exact config. Store data in _chartData for getChartConfigs().

5. **Tiles block**: 2x2 grid:

   - Largest 24h: max magnitude from live data

6. **Explanation block**: 2 paragraphs via i18n about plate tectonics and earthquake measurement scales.

7. **Comparison block**: Brief comparison text about earthquake frequency globally.

8. **Sources block**: USGS Earthquake Hazards Program with link.
