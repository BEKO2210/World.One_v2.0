Use growth rate 0.9% (from static values). Calculate per-second increment: `perSecond = population * (0.009) / (365.25 * 24 * 3600)` which is approximately 2.3 people/sec.

   Create the counter display element, then start a setInterval(1000) that updates the number. CRITICAL: Push the interval ID to `_intervals` for cleanup.

   ```javascript
   const latestPop = data.total[data.total.length - 1].value;
   const perSecond = (latestPop * 0.009) / (365.25 * 24 * 3600);
   const startTime = Date.now();
   const counterEl = DOMUtils.create('div', { className: 'pop-counter', style: { fontSize: '2.5rem', fontWeight: '700', color: 'var(--text-primary)', fontVariantNumeric: 'tabular-nums' } });

   const update = () => {
     const elapsed = (Date.now() - startTime) / 1000;
     const current = Math.round(latestPop + elapsed * perSecond);
     counterEl.textContent = MathUtils.formatNumber(current);
   };
   update();
   _intervals.push(setInterval(update, 1000));
   ```
